# CEAP

Client Enagement Analytics Portal,客户参与数据分析的平台


# 技术栈

## 后端

spring cloud gateway
spring cloud security
spring boot3
java17
Nacos
Dubbo

## 前端

antd pro

react

echarts 图表


# 项目结构

ceap: java后端项目
admin: 管理平台（PC端）
app: 小程序（手机端）

# 实现功能

1. 管理员账号的查询、修改密码、添加、更新、停用
2. 权限规则的查看
3. 角色的查看、添加、添加权限、删除权限、修改角色名称
